== Installation ==

1. Download the plugin from [Official WP Plugin Repository](https://wordpress.org/plugins/bank-transfer-select-for-woocommerce/).
2. Upload Plugin from your WP Dashboard ( Plugins>Add New>Upload Plugin ) the bank-transfer-select-for-woocommerce.zip file.
3. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
